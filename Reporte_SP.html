Introduccin

La prediccin del desempeo acadmico de los estudiantes es un tema de gran relevancia, ya que permite identificar factores que influyen en las calificaciones finales y orientar estrategias de apoyo educativo. En esta actividad, se utiliza una base de datos de 395 estudiantes que incluye informacin demogrfica, acceso a internet, historial acadmico y calificaciones de los dos primeros periodos del curso. Transformacin de variables categricas

Antes de aplicar un modelo, es imprescindible que todas las variables de entrada sean numricas y adecuadamente codificadas. Las variables categricas deben ser transformadas para que el modelo pueda interpretarlas correctamente.

En esta seccin se realiza la codificacin de las variables categricas (Escuela, Sexo, Internet) mediante la tcnica de one-hot encoding, generando variables dummy binarias que representan la presencia de cada categora, evitando redundancias mediante drop_first=True.

Adems, se presta atencin a variables que, aunque parecen cuantitativas, en realidad representan clases discretas. Este es el caso de HorasDeEstudio, donde los valores numricos indican rangos de tiempo de estudio en lugar de magnitudes continuas. Para capturar esta informacin de manera adecuada, se cre una codificacin binaria de dos variables:

  • estudioMedio: indica si el estudiante dedica de 2 a 5 horas de estudio.
  • estudioAlto: indica si el estudiante dedica ms de 5 horas de estudio.

Con esta representacin, cada estudiante queda codificado de la siguiente manera:

estudioMedio estudioAlto Interpretacin
0 0 Estudio Bajo (<2 horas)
1 0 Estudio Medio (2-5 horas)
0 1 Estudio Alto (>5 horas)

Esta codificacin permite que la variable sea binaria y interpretable por el modelo de regresin, evitando ambigedades que podran surgir si se usaran los valores originales como enteros continuos.

Identificacin de valores atpicos en la variable "Faltas"

La presencia de valores atpicos puede afectar significativamente la estimacin de los coeficientes en un modelo de regresin lineal mltiple.

Para detectar observaciones atpicas en la variable Faltas, se aplic el mtodo de Tukey con un factor k=3, utilizando el rango intercuartil (IQR). Este criterio considera como outliers aquellas observaciones que se encuentran a ms de 3 veces el IQR por debajo del primer cuartil o por encima del tercer cuartil.

In[4]:
import numpy as np

q1 = df["Faltas"].quantile(0.25)
q3 = df["Faltas"].quantile(0.75)

iqr = q3 - q1

li = q1 - (3 * iqr)
ls = q3 + (3 * iqr)

out = np.where((df["Faltas"] <= li) | (df["Faltas"] >= ls))[0]

print("Observaciones atpicas en 'Faltas':")
df.iloc[out, :]
